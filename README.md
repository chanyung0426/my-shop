# REACT - my-shop Project

리액트 쇼핑몰 프로젝트

## 🖥️ 프로젝트 소개

firebase를 통한 연습용 리액트 쇼핑몰 사이트입니다.

### ⚙️ 개발 환경

- `Javascript`
- `HTML`
- `CSS`
- **Framework/Library** : React
- **Database** : Firebase

## 📌 주요 기능

-상품등록 페이지 제작(관리자 권한)
-상품 등록해서 파이어베이스 데이터베이스에 저장
-로그인 기능 구현(소셜 로그인, 일반 로그인)
-관리자 아이디 구현(특정 권한 부여)
-카테고리 구현
-카테고리별 리스트 제작
-검색창 구현
-상품의 상세페이지 제작
-장바구니 구현
-QnA 페이지 - 일반과 관리자 구별
